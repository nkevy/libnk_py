===========
Nknet
==========
The Nknet library contains python modules to preform network analysis.
A typical usgae might look like the following example:

	from nknet.Nettool import Nettool
	
	nt = Nettool()
	ip = nt.get_local_ip()
	ip_list = nt.get_all_local_ip()
	
(note: The example does not follow python indetation conventions and thus is only a model)
