==============
libnk_py
=============

The libnk_py repo holds packages for python.

To install packages:

	1. Clone the repo:
		git clone https://github.com/nkevy/libnk_py.git

	2. Navigate into the required package, for example:
		cd Nkecon

	3. Add the package to pip:
		sudo pip install -e .
